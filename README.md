# WireShark_pcapng

В этом репозитории содержатся файлы захвата для анализа и обучения работы в программе Wire Shark. Вы можете использовать их по своему усморению, но, по возможности, оставляйте ссылки на данный репозиторий.

Здесь Вы также найдете файлы: `lotsofweЬ.pcapng`, `wrongdissector.pcapng`, `http_google.pcapng` которые упоминаются в книге "Анализ пакетов. Практическое руководство по использованию WireShark и tcpdump для решения реальных проблем в локальных сетях." автор: Крис Сандерс.
